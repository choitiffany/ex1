User Needs

    Needfinding for nomore.com

  • Needs a way to organize information better.
  • Needs a way to improve language so that it's not confusing to the user (Learn vs. About)
  • Needs a way to help users navigate and get a better sense of direction on the site
  • Needs a way to clearly show how to help
  • Needs a way to not overwhelm the user with information.
  • Needs a way to clearly indicate where the user can click
  • Needs a way reduce the various components of the site
  • Needs to be able to present an end solution to the user
  • Needs a way to easily access key information about the campaign
  • Needs to be able to assure the user that things will be okay

I used my roommate for user research. Some notable things she mentioned was that there were "too many things jumping out at me". She studies human development and mentioned how if she were a victim, she wouldn't know what to do on the site. She mentioned how there needs to be an emphasis on how to help (for non-victims) and a way to assure victims that things will be okay.
